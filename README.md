# Python-Functions-101


## Building code

### Step 1: Create and source a virtualenv so libraries are easy to use

```
virtualenv ~/.venv
source ~/.venv/bin/activate
```

### Step 2: Create the basic scraffold

```
touch Makefile
touch hello.py
touch requirement.txt
```

### Step 3: Create a unit test

```
touch test_hello.py
```
